Our Way of Thinking

We believe in robust, extensible, and well-designed products. That’s why we created Novæ. Our goal, as a team, is to deliver the product that fits your needs. Don’t worry about the technical details – we will handle that for you.

Solid and Extensible Architecture

No matter the size of your project, it will grow – maybe really fast.
Solid foundations and a scalable architecture are required to keep pace while traffic increases. We care about providing technologies and building software that will go with you along the path.

State-of-the-Art Technologies

Ruby on Rails is a solid and mature framework, whose goal is to deliver fast and stable web applications. We use it to create good software in no time.

NodeJS is the uprising Javascript framework that lets us build fast and event-driven data API, allowing us to code in the same asynchronous philosophy in frontend and backend.

AngularJS permits the creation of reactive user interfaces and uncouple views from data API logic. Applications become fast and easy to maintain.

C++ enables us to unleash every single bit of power from the CPU, to create large and powerful applications.
